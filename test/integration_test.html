📝 新的Key格式範例

現在支援的格式：

Key: Bb (AG:G)   Capo: 3

說明：

  • Bb = 目標調性（演出調性）
  • G = 原調性（AG，吉他實際彈奏的調性）
  • 3 = Capo位置

載入後的預期狀態：

  • 原調: G ✓
  • 目標調: Bb ✓
  • Capo: 3 ✓

🔄 向後相容性

舊格式仍然支援：

Key: G   Capo: 0

會被解析為：原調=G，目標調=G，Capo=0
